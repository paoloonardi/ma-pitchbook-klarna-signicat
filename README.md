# M&A Buy-Side Pitch Book — Klarna / Signicat (Illustrative)

Independent valuation and deal-structuring exercise, built to demonstrate
IB-style analysis alongside a quant/tech background.

**Deal:** Klarna Group plc (NYSE: KLAR) — hypothetical acquisition of
Signicat AS (Nordic Capital-owned), Nordic identity/KYC platform.

**Scope:** industry landscape, trading comparables, precedent transactions,
football field, accretion/dilution, deal structure, implementation timeline.

**Note:** hypothetical transaction constructed from public company data,
for CV and interview discussion purposes only. Not a real proposed deal
and not investment advice.
